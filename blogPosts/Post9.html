Visualizing Graph-Related Attributes of a Financial Network

By Tamer Khraisha

Ph.D , Department of Network and Data Science, Central European University (Budapest)

The following D3 chart is a temporal visualization of 24 attributes of a financial network. The network has banks as nodes and a link exists between two banks whenever they partecipated together in granting a loan. The temporal window is from 1984 until 2011. The size of each circle indicates the normalized value of each attribute where normalization is done with respect to the sum over all years. To see the values in numbers you can hover the mouse over the attribute in the legend.
